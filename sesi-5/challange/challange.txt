Buatlah sebuah aplikasi Spring Boot yang menggunakan JPA dan Hibernate untuk mengelola data produk dalam basis data. Gunakan MySQL atau PostgreSQL sebagai basis data dan jelaskan perbedaan utama antara keduanya. Optimalkan query dengan menambahkan indeks pada kolom yang sering digunakan, menerapkan transaksi untuk operasi CRUD, dan normalisasi tabel hingga Third Normal Form (3NF). Implementasikan repository pattern dengan clean code principles, termasuk penggunaan interface untuk repository, pemisahan logika bisnis dari logika akses data, dan penamaan metode yang jelas. 
Output goals yaitu berhasil menerapkan JPA dan Hibernate hingga masuk ke database.
